# Читака

## Описание
Многостраничный веб-сайт "Читака". 

## Структура проекта
```
/Читака
  /data
    books.json                  ← массив книг: id, title, author, description, shortDescription, genre, rating, cover
  /images/books/                ← обложки
  /pages/
    index.html                  ← Главная (без авторизации)
    index-auth.html             ← Главная (с авторизацией)
    login.html                  ← Вход
    register.html               ← Регистрация
    search.html                 ← Поиск
    favorites.html              ← Избранное
    profile.html                ← Профиль
    book.html                   ← Карточка книги
  styles.css                    ← Стили
  script.js                     ← Функциональность
```

## Запуск
Инструкция по запуску сайта "Читака"
  Перейдите в директорию проекта.
  Запустите 
  ```
  python server.py
  ```
Сайт будет доступен локально по адресу http://localhost:8000/pages/. Убедитесь, что запускаете сайт через HTTP-сервер, а не открываете HTML-файл напрямую, чтобы корректно загружались данные из books.json.
